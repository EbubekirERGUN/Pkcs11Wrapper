# Code of Conduct

## Our standard

We want this repository to be a constructive, respectful place for technical collaboration.

Please:

- be respectful and direct
- critique ideas, not people
- provide actionable technical feedback
- avoid hostile, dismissive, or personal attacks

## Scope

This applies to:

- issues
- pull requests
- code review discussions
- repository-related communication

## Enforcement

Project maintainers may moderate or remove clearly harmful behavior to keep collaboration productive.
